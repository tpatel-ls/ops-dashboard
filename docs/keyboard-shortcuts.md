# Keyboard shortcuts

Press `?` anywhere to open this list as an overlay (lands in M1).

## Global

- `Cmd K` open search
- `g then i` open quick add
- `Cmd Enter` save the active form
- `Esc` close the active overlay

## Navigation

- `g then t` Today
- `g then w` Week
- `g then m` Month
- `g then c` Calendar
- `g then k` Kanban
- `g then b` Whiteboards
- `g then n` Inbox

## Task list

- `j` and `k` move selection
- `x` toggle done
- `e` edit title
- `1` to `4` set priority
- `t` then a date snippet (today, tomorrow, mon, etc.) reschedule
- `#` add a tag inline

## Whiteboard

- `v` select tool
- `p` pen
- `e` eraser
- `r` rectangle
- `o` ellipse
- `Cmd Z` and `Cmd Shift Z` undo and redo
